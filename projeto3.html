Sobre o projeto

Este projeto simula uma loja virtual completa: uma landing page de apresentação conectada a um sistema de back-end em PHP, com autenticação de usuários, cadastro de produtos e fluxo de recuperação de senha, tudo persistido em um banco de dados MySQL.
