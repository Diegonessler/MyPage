Sobre o Projeto

Desenvolvi, individualmente, um projeto acadêmico inspirado no Spotify, com foco em funcionalidades como criação e gerenciamento de playlists, busca de músicas e perfis de usuário. A aplicação foi projetada para oferecer uma experiência fluida e intuitiva, com navegação eficiente entre telas. No front-end, organizei as interfaces com ênfase em usabilidade, garantindo um design moderno e responsivo. No back-end, utilizei Node.js para construir uma API robusta, integrada a um banco de dados que armazena informações de usuários, playlists e músicas. Implementei um sistema de cadastro e login com criptografia de senhas para maior segurança, além de um mecanismo dinâmico que atualiza a exibição de músicas com base em sua popularidade. Durante o desenvolvimento, enfrentei desafios técnicos, como a integração entre front-end e back-end e a otimização da navegação, que foram superados com soluções criativas e boas práticas de programação
